함수

1. 선언적함수 

        function 함수명([매개변수, 매개변수, ....]){
            함수호출시 실행할  코드 ; 
            [return 결과값; ]
        }

2. 익명함수 (이름없는 함수) 

        변수| 속성 = function(매개변수 ) {
            실행할 코드;
            [return 결과값;]
        }
        특정 변수나 속성에 대입되는 함수 제시시 주로 사용됨 
        (주로 이벤트 핸들러 작성시 사용) 
        이벤트가 안먹힐 수 있다 완전하진 않고 잘 사용하지 않는다.

3. 화살표 함수 (arrow function)

        - 본문에 리턴만 있을 때
          변수 = ([매개변수, 매개변수,...]) => 리턴값;
        - 본문에 내용이 여러줄 일 때 
          변수 = ([매개변수, 매개변수,...]) => {     
            한줄일 때는 한줄로,  넘어감 
            실행코드 ;
            실행코드 ;
            ...
            [return 결과값;]
          } 
        - 익명함수를 간단히 작성하는 방법 (ES6(ES2015)에 추가 )

3 - (1) 매개변수가 없는 화살표 함수 실행

3 - (2) 매개변수가 1개인 화살표 함수 실행

3 - (3) 매개변수가 2개 이상인 화살표 함수 실행

4. 즉시 실행 함수 (iife(Immediately Invoked Function Expression)) 

        - 정의하면서 즉시 실행되는 함수
        - 단 한번만 호출됨
          ([변수] = function([매개변수, 매개변수,...]) {
            실행될 코드;
          })(인자,....);
        - 실행 확인 : F12누르고 - console 부분에 나옴
