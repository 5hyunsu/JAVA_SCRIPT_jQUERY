변수와 자료형

변수 선언시 var뿐만 아니라 let, const이용하여 선언 가능 (권장사항)

*var, let, cons의 차이점

1) 변수 선언 방법

2) 스코프 (사용 가능한 범위)

* 변수 선언 위치에 따른 전역변수 / 지역변수

* 자료형

Dynamic Typing : 변수 선언시 자료형을 별도로 지정하지 않음(자료형의 개념이 없는 것은 아님) => 변수별로 타입이 지정되지 않고, 대입되는 값에 따라 타입이 결정됨

자바스크립트의 자료형

  • string(문자열)
  • number(숫자,실수,양수,정수 다 )
  • boolean(논리값)
  • object(객체)
  • function(함수)
  • null (초기화가 안되어 있는 변수)
  • undefined (정의되지 않은 변수)
